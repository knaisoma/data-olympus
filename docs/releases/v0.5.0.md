## New features

- Data Olympus can now tell you when a newer version has been published. A small,
  fail-safe check compares the version you are running against the latest release
  and produces a one-line "a newer version is available" notice, so deployments
  that do not auto-update can tell at a glance that they are behind. (Surfacing it
  in the running server and MCP responses is landing across this release.)
