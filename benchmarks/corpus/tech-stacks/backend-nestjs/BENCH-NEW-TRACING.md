# tracing

This concept defines the governance for tracing. When working with tracing, follow the rules below. The tracing approach affects monitoring, rollback, guardrail, review.

- Prefer the documented tracing pattern.
- Record exceptions to the tracing rule.
- Weigh the monitoring and review implications.
